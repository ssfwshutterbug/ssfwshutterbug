zsh

ZSH

change default location

.zshrc and .zcompdump location in $HOME directory. and we can change it to another folder

# add this to ~/.zprofile
export ZDOTDIR=$HOME/.config/zsh

Note:

.zcompdump file used to speed up the zsh completion

change oh-my-zsh default location

export ZSH=$HOME/.config/zsh/.oh-my-zsh

auto correct

In archlinux installer, i have ever saw auto correct effect while error typing and it is very helpful, however i cannot find the plugin which has the same effect. Today, i searched and found it finally, you know what? It is even not a plugin but a configuration in .zshrc.

# uncomment to enable it
ENABLE_CORRECTION="true"

  • n which means no, don’t correct my command and execute it as it is
  • y which means yes, correct my command
  • a which means abort, don’t execute the command and give me an empty prompt
  • e which means edit, don’t execute the command and let me edit it again

auto suggestion

oh-my-zsh includes this function. Add zsh-autosuggestions to plugin{} will okay. However, it is much more customizable, we can even define its color.

plugins=(git
    zsh-autosuggestions
    )
ZSH_AUTOSUGGEST_HIGHLIGHT_STYLE="fg=#7e749d,bg=#42414d,bold,underline"
