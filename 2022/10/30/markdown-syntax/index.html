MARKDOWN SYNTAX

There is no rules about how many lines we should put between paragraph, but basically we need 1, if we put more than 1, it is just for good view.

  • syntax character has a space with normal document.

  • use one or more lines create paragraphs.

  • use 2 or more spaces break a line

  • use 1 * to italicize text

  • use 2 * to bold text

  • use > to create blockquotes

  • use number followd by period to create orderd list

  • use - to create unorderd list

  • add elements in lists with 4 or a tab space in the beginning of line

  • use ` to denote a word

  • use 4 spaces or a tab to create code blocks

  • use 3 or more - create a line

  • use to create a link

  • use to create a image

  • reference-style links
