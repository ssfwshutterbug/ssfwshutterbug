USE MACRO

If we don’t know macro in vim, we have no idea how powerful vim is. Macro allow us do some operations(normal or insert or ex command) then replay it. We start macro with q then press a character which stores the macro, end macro with q.

After record, we can use @x (example x is the register) to replay. A number can be set before @x which means replay it several times.

What if we miss something in the macro record, should we do it again? No, we can edit macro in two ways:

  1. paste macro in vim, edit and copy to register again.

  2. paste macro in ex mode, and edit(i like this way, since some none visible characters will show in ex mode).

Recursive Macro

Vim supports recursive macro, that is amazing. With this feature, we don’t need to specify how many times to run macro again. Because in most of time, we don’t know exactly times we need to run it again. The trick is in the end of record pressing @x (example x is the register). Then this macro will run forever until the file ends.

DELETE TO REGISTER

Instead of deleting text to default register " with d.., we can delete to a register instead. The benefit of register is we can paste those deleted text again. Just think about it, we now have 26*2 standalone clipboard.

Delete to a register: "register d.., paste with ctrl+r register in insert mode. ctrl+a will insert last inserted text.

Default useful register:

register content
% current file name
/ last search pattern
: last command line
- last small delete

MOTION-UNIT-POSITION

One magic of vim operation is that we can combine motion-unit-positon to do lots of stuff.
such as d stands for delete motion, but what should be deleted? Okay, we use unit and position to specify. d2j means delete next 2 line, dip means delete in a paragraph. de means delete to a word end.

UNIT

unit includes numbers and a or i.
a means a/an, i means in. The magic of a or i is we can do something in a line without current cursor position exactly under the one we operate. For example we can in one line’s beginning but delete a word which is surrounded by “” in any position of the line with da" or di"

  • daw: delete a word
  • di": delete in “”
  • di': delete in ‘’
  • dis: delete in sentense
  • yaw: copy a word
  • yas: copy a sentense

SPELL CHECK

Enable/Disable spell check

There are 3 state of a word:

  • Good: real good word
  • Misspell: not sure
  • Bad: read wrong spell word

We can mark word good or bad, and also can save it to file or as tmp.

TIPS

MULTI INSERT

shift+v :norm I char: add some char in multi lines’head

ctrl+v :I char ESC: add some char in multi lines’specified position

NUMBER SEQUENCE

shift+v g ctrl+a: number sequence, line head

ctrl+v g ctrl+a: number sequence, specified position

REVERSE LINE SEQUENCE

:g/^/,m0: reverse line sequence, move matched line to zero line

JOIN MULTI LINE

J: join two lines, with one blank space
