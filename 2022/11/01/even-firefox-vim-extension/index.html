EVEN THE FIREFOX VIM EXTENSION

Yeah, this topic may seems a little wired, it is just a simple firefox extension called Vimium C, and i have to talk about it. Okay, i want to say it is not my problem, it is the extension itself makes thing awful, it has nearly zero key binding example, how do i know how to configure it? In that way i have to record some important things before i can get used to it.

By default K moves to next tab and J moves to previous tab, the logic makes me uncomfortable, it is different from VIM. Another is no or i don’t know the key of scroll page down/up.

So in its configuration page, i need rewrite this kind of key bindings.

caret mode is something we can show mouse position and scroll in lines, with v key pressing, can select words or lines.

There is a option unchecked by default must be checked, and it is “Don’t let pages steal the focus in loading”. It can help us move between tabs when page is loading instead of useless waiting.
