CLEAN HTML TAG IN CSV WITH VIM

Today, one people ask me whether azure devops platform can export csv file with clean plain text. Because one column in csv fulls of html tags. However there is no such a function.

I know there are some other methods, such as write a python script. But it is not easy to build a python environment especially in company. And it is really too complicated, if we use vim to clean those tags, that’s really simple.

There are some tags we need to clean:

  1. clean end of tag such as </div>
  2. clean all info in <>, such as <span ...> <a herf="...">

Tips:
There is a little trick, when we match any content in <>, we need to use none greedy
regex, we know in standard regex, we can use ? to specify. But in vim, we need to
use \{-}, which means match previous atom 0 or more times as less as can.

And there is another needs: if the content in one cell can be in multi lines, that will much better. And in origin html, a <br> tag means line break. So the problem becomes how to replace <br> tag to a new line symbol in vim.

After searching, i find the way to type line break symbol in vim, that is typing ctrl+v ctrl+m in insert mode, then it will display a line break symbol, when we open the csv file in other excel programs, it will shows multi lines in one cell.

But, it is really hard for other people to type those commands, so i figure out the way, using vim’s autocmd. The user only needs to use vim to open the file, then type :wq, everything will be done.
