OPERATE SPECIFIED COLUMN IN VIM

I got a task which one people asks my help to clean up some text in a csv file. But it seems not an easy work, i mean in vim. Because vim doesn’t has the ability to operate on some specified columns, which awk or pandas can do it better.

Example: the column separates by ",", i need to change column two’s text between {} to ===, and change column three’s text between {} to +++.

At first, i tried use vim autocmd to do that work, but i failed and then i tried to use awk and sed and paste. However in the real text, i need use none greedy regex to match, and sed doesn’t has the ability. So i completely failed.

And then i tried vim again. I wonder whether vim has the ability allows me to match ",", then just like in normal mode presses vn to select the second column, and then types :s/{\a*}/{===} to replace the text.

I tried use global search plus norm command:

But i just can not like manual operation to visual select those text.

I need to replace selected text, i get another problem: every time i replace text, i can only replace one character. I cannot reselect those text. After googleing, i know the solution: we need to use \%V to stand for visual selected text.

But it is just not working properly, After googleing, i finally know the solution.

  1. use exe at the beginning.
  2. we can combine ex command with norm command, but need to wrap norm command in '' to be string.
  3. in norm command, we need to press return and escape just like manual type, so we need use ctrl+v to help us to input those symbol.
  4. after execute norm command, if we need to execute ex command, we need to use | to separate them.

CONCLUSION

  1. use exe to start ex command.
  2. wrap norm command in '' to be string with ex command.
  3. norm command cannot followed by ex command except add a | at the ex command begin.
  4. reselect visual selected area, we need use \%V.
  5. we can write special character such as escape, return with ctrl+v pressed.
