blender learning

blender

shortcut

shortcut function
g grab
r rotate
s scale
left click choose object
middle click change view port
right click context menu
` view
a select all
aa not select all
shift+a quick add menu
F3 quick search
Tab object mode/edit mode

combine use

  • click one object and type g/r/s then press x/y/z to choose x/y/z axis
  • click one object and hold middle mouse button to snap to x/y/z axis
  • click one object and type g/r/s to get some action then click left mouse button to finish the action, click right mouse button to undo the action

some principles

  • while modling, simple segments much better than complex segments
  • while modling, adjust segments to make each plane similar to square
