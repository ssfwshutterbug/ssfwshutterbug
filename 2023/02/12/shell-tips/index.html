SHELL TIPS

This file is used to collect some useful shell usage.

EXECUTE SUB SHELL COMMAND

Sometimes, we are in a directory A, we want to execute a command in directory B, and then we want get back to directory A. In a fact, we don’t need to change directory twice, we can do it in sub shell.

For example, it is pretty useful when using with Hexo.

use $_

In bash, $_ stands for previous command’s path, if there has one. So with this variable, we can type easily.

use `!!`

maybe there is no variable to store last command output which is very useful in terminal usage, but we can get that with `$()` or `!!`

the same as: du -sh /nix

user$ du -sh $_

It works fine, but what if we need the original ls or cat, we don’t need to write /bin/ls, we can easily use \ls, it means the original ls, not the alias one.

view a bz.2 file directly

xx.bz2 doesn’t need to be extracted, we can view it’s content directly with bzcat. Most of the time we view a bz2 file with linux doc file.

Delete Link To A Directory

I got a problem, a link Pictures linked to my home directory, so i cannot rm -rf Pictures and unlink Pictures/ also failed.

The problem is caused by training slash after link name. When i removed the slash after, it works.
