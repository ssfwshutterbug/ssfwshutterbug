SUCKLESS STUFF

Yeah, i recently start using gentoo, and use suckless dwm+st+dmenu. There are some important things need to be documented.

PATCH

As we know, suckless tool is not good to use by default, we need to patch, and which patch should we choose is our own decision.

  • Download dwm patch (save the diff file) for example

  • Start patching

    Note: By default, make will read config.h file to compile, we can edit config.h file directly, and in this way, we don’t remove config.h file. But i prefer to use config.def.h, so every time editing and before make, i need to remove config.h.

  • Fix error

    If we luck enough, we will patch without errors. But most of the time, we will meet errors. So fix error is a very important skill. And keep in mind, it is really easy.

    If error appears, there are three files we need pay attention to:

    1. config.def.h: unfinished patch file, contains some modifies but not complete, we need to edit by hand.
    2. config.def.h.rej: show us which parts of config.def.h should be edited by hand.
    3. config.def.h.orig: the origin unedited config.def.h, the backup of config.def.h.

    So now we need to modify by hand. We can use neovim to open the two files in the same time, convenient for us to copy and paste.

  • Compile

    After patching and fixing error, it is time to compile.

    Tips:
    we can use make clean install to instead of above three commands.
    reverse/undo patch with patch -R < xx.diff.

THE PATCH I USE FOR NOW

  • FOR DWM

    1. DWM-FULLGAPS-TOGGLE-20200830.DIFF (ADD GAPS BETWEEN WINDOWS)
    2. DWM-KEYCHORD-20211210-A786211.DIFF (SUPPORT MOD+KEY +OTHERS)
    3. DWM-MAXIMIZE_VERT_HORZ-20160731-56A31DC.DIFF (ADD MAXIMIZE FUNCTION, GREAT)
    4. DWM-STICKY-2022-12-05-D47BA0B.DIFF (COPY WINDOW TO ALL TAGS)
    5. SHIFTVIEW.C (SUPPORT KEYPRESS TO CHANGE TAGS, FOR EXAMPLE USE MOD+H/L TO CHANGE TAG, CONVENIENT)

  • FOR DMENU

    1. DMENU-FUZZYMATCH-4.9.DIFF

  • FOR ST

    1. ST-SCROLLBACK-RINGBUFFER-0.8.5.DIFF
    2. ST-SCROLLBACK-MOUSE-20220127-2C5EDF2.DIFF
    3. ST-SCROLLBACK-MOUSE-ALTSCREEN-20220127-2C5EDF2.DIFF

SOME CONFIGURATIONS

how to change st line space?

We can change the number in st config.def.h to adjust line heigh

how to make st’s bottom space normal?

By default, st’s window size will change somehow, we can edit the configuration of dwm to make it work normal or patch dwm with st-anysize-20220718-baa9357.diff

DWM STATUS BAR

Not like xmonad needs a standalone bar which can be xmobar or polybar, dwm don’t need to install an extra bar, itself contains a one. And the way to configure it is really funny and limited in my opinion.

We can use a command to set the information which we want to display in bar.

For example show time in bar.

So, i write a simple shell script to display some info.

And run this script when startx. Write to ~/.xinitrc
