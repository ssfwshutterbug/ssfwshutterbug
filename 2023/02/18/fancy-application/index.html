Fancy Application

My favorite application lists.

FreeRDP

A replacement of rdesktop which supports connecting windows with rdp protocol, since rdesktop doesn’t support new windows’s secure policy and stop maintenance, i now use FreeRDP, yet another beautiful application.

Urxvt

Ah, that really goes years since last time i use urxvt, with time goes by, i use some kind of modern terminal emulator like alacritty and leave urxvt behind. But when i turn to gentoo linux, all those old, efficient but simple tools remind me again, because the others really contain lots of extensions need to be built and i even don’t use or heard that function or ability.

Yeah, xterm is also a good choice, it is also very simple and just work fine, it can get lots of my stuff done. But i just wanna to try urxvt, it gives me some abilities which xterm doesn’t have and i really want, such as copy the output via keyboard.

Another important thing is i finally know that most of terminals have a daemon, which really helpful for faster launch speed and less memory usage.
