KVM

In this topic, i will focus on kvm in gentoo linux. KVM stands for kernel virtual machine, itâ€™s a feature of linux kenel. It supports all needed api, but only. So if we want to create a real virtual machine, other software needed.

  • qemu
  • libvirt
  • virt-manager

Install

qemu

If we only install qemu, we can create and run virtual machine. But it will lack of some features.

libvirt

Libvirt supports the additional features, however we cannot use virt-install to create virtual machine, and we have no GUI by default.

virt-manager

Finally, virt-manager will give us all we need, but one thing need to be noticed, if we donâ€™t build it with gtk use flag, we cannot launch it neither from terminal nor app launcher such as dmenu.

Usage

qemu

Once we installed qemu, we can use qemu-system-x86_64 to launch virtual machine.

libvirt

When we installed libvirt, we will get virsh command.

As we said before, there is no virt-install package for us to create new virtual machine if we donâ€™t have virt-manager installed. But we do can create new machine with qemu-system-x86_64 and then configure and launch with virsh.

All libvirtâ€™s configuration is depend on xml file, if we already have a qcow2 image, we can edit its xml file to import and run this machine.

GUI

If we run machine with qemu-system-x86_64 or virt-manager, we already have a gui display.

If we run machine with virsh, we donâ€™t have a GUI display, we need other remote tools to help us to connect.

For connecting of windows, we can use freerdp, the better version of rdesktop.

Issue

Video card cannot enable 3D acceleration

It is because qemu has not built with virgl flag.

So, we can enable this flag and rebuild qemu.

Virtual Network Interface cannot select virtio

You must enable VHOST_NET to have vhost-net support
