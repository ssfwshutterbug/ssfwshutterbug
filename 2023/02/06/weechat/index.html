WEECHAT

Weechat is a text-mode IRC client, we can talk to people through channel and people where they come from the whole world, it is amazing!

A few years ago, i have used weechat, and now i want to use it again. But i almost forget how to use it, so this article is mainly focus on how to get things done.

install weechat in gentoo

basic configure weechat

With the following steps, we can simply configure weechat, and join some channels.

  1. start weechat

  2. add irc server

  3. set nickname

    Note:

    After this step, we can connect to libera server and join free channel.

    Next time we start weechat, we only need to follow steps showing below.

  4. connect to server

  5. join channel

  6. close channel

  7. quit weechat

registering account on the Libera.Chat IRC network

If we only do the configuration above, we may failed to join some channel such as #archlinux-aur, cause it needs a registered NickServ account logged in.

So we need to register an account on Libra.Chat, it can be done through command(make sure we connect to libera server and then type the command).

Note:

password is the one we need to remember for the next time to login account

email should be a in use email account, cause we will receive an email include the information about how to finish the register process.

Now we need to open our email web or app to open the new email which is sent from Libera.Chat Network Services, find the command which tell us how to run this in weechat, we need to copy this command and run in weechat to finish register.

Run this in weechat, this command is random, so don’t use this one next time.

Now we get a registered NickServ account, we can join #archlinux-aur channel, before join, we need to vertify.

And we are allowed to join.

enable auto connect

Although we can connect to libera server by hand, there is a convenient way to auto connect.

Yeah, there is also a way to auto join channels, but so far, i don’t like to auto join channel, may be in the future i will use this.

shortcut

Alt+1 switch to channel 1
Alt+2 switch to channel 2
…

Tips:

  1. We can type command in every channel, the weechat will recognize it as command but not sending message. So no need to worry about where to type command.
