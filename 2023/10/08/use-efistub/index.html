USE EFI STUB

I used to use grub as my bootloader, and it works great. But i wanna to try something new, so this time i use EFI stub.

In a fact, Grub is a secondary bootloader, because UEFI fireware cat act as a bootloader, it stores boot information in NVRom, so we can directly boot linux kernel via UEFI. The only tool we need is efibootmgr which can help us to configure boot information in NVRom.

Assume nvme0n1 is the disk, nvme0n1p1 is ESP partition and nvme0n1p2 is root partition.

install tool

prepare efi boot entry

We need create a efi directory under / and mount ESP partition under /efi.

Then create directory and copy kernel boot entry to folder.

create UEFI boot menu

Now we can create a UEFI boot menu with efibootmgr. According to this article, there are two options for us to choose to specify the root partition.

  • configure root partition path to kernel.
  • configure root partition path to UEFI.

I choose the second method, cause it is super easy, only need to add one more parameter -u to efibootmgr command.
