JS

Concept

  • DOM: document object model. Represents all page content as objects that can be modified.
  • BOM: browser object model. Represents additional objects provided by the browser for working with everything except the document.

Notice:
both are object, means all elements are objects which have method or function.

For example: if we want to know browser and used platform or url, it belongs to BOM.

DOM Tree

DOM tree shows how the page constructs. Root is html and leaf is always text.

querySelector(css) vs querySelectorAll(css)

querySelector(css) returns the first element for the given CSS selector. querySelectorAll returns all elements. So querySelector is faster than the other.
